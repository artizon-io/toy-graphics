# Toy OpenGL

A toy OpenGL project

## Setup

Dependencies
- `Cmake` for generating builds
- `Conan` for installing libraries
- `OpenGL` support on your system

1. Navigate into `Milestone_x`
2. Give the following 3 scripts execute permission
3. Run `./configure.sh`
4. Run `./build.sh`
5. Run `./execute.sh`
